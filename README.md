# Vertex Shift Method (VSM)

This repository contains the implementation and experiments for:

"The Vertex Shift Method: Polynomial Normalization, Spectral Translation, and Dynamic Extensions"

## Contents
- VSM implementation notebook
- Numerical experiments
- Reproducibility setup

## Description
VSM is a derivative-based normalization technique that eliminates the linear term of a polynomial by shifting to a critical point P'(φ)=0, improving numerical conditioning in root-finding problems.
